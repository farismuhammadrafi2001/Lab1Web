Mengenal Dasar Bahasa Pemrograman

disini KIta memakai code editor Sublime Text 3, Karena Lebih Muudah Dipahami untuk orang yang baru belajar bahasa pemrograman, HTML Adalah HyperText Markup Language (HTML) merupakan suatu bahasa markah yang digunakan untuk membuat halaman web, menampilkan berbagai informasi di dalam sebuah browser web. HTML dibuat dengan menggunakan aplikasi pengolah kata dan disimpan dalam format ASCII sehingga menjadi halaman web dengan perintah-perintah HTML.
